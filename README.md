# 🗺️ **BooMap**

---

🧭 _당신의 프로젝트에 길잡이가 되어줄게요_ 🧭

**AI 기반 마인드맵 자동화 서비스**

> 💡 _회의만 하면 마인드맵이 자동으로 만들어진다고? 히트다 히트!_
>
> - 회의록 기반으로 **마인드맵을 자동 생성**해요.
> - 시각화된 캔버스에서 **수정과 삭제**가 가능해요.
> - 만들어진 마인드맵을 팀원들과 **쉽게 공유**할 수 있어요.
